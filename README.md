##  Descrição

Este projeto é um sistema bancário web simples que permite ao usuário visualizar e gerenciar suas finanças de forma prática.

###  O que o sistema faz:

* Visualiza saldo da conta
* Exibe extrato de transações
* Mostra entradas e saídas
* Apresenta gráficos financeiros (mensais e gerais)
* Pode fazer transações para outra conta por meio de email

###  Linguagens e tecnologias:

* **React** → interface do usuário
* **JavaScript** → lógica da aplicação
* **CSS** → estilização
* **Node.js** → backend
* **MySQL** → banco de dados

###  Por que esse projeto foi feito:

* Praticar desenvolvimento fullstack
* Simular um sistema bancário real
* Aprender integração entre frontend e backend
* Trabalhar com gráficos e manipulação de dados

##  Instrução de instalação

###  Pré-requisitos:

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)
* Banco de dados MySQL configurado

---

###  Passos para rodar o projeto:

#### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

#### 2. Acesse a pasta

```bash
cd frontend
```

#### 3. Instale as dependências (frontend)

```bash
npm install
```

#### 4. Inicie o projeto

```bash
npm run dev
```

---

###  Backend

```bash
cd backend
npm install
npm run dev
```

Configure também:

* conexão com banco de dados (MySQL)
* variáveis de ambiente (token, porta, etc)

---

## ▶ Instrução de uso

1. Acesse o sistema no navegador
2. Faça login (Caso não tenha faça login)
3. Navegue pela página principal
4. Visualize:

   * saldo
   * extrato
   * gráficos
5. Utilize as funcionalidades disponíveis (como transações)
